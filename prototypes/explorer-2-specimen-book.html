One stylesheet, set as its own manual.

Ink on paper, hairline rules, hard corners, one font size on every surface. Every plate below is rendered live by the file it documents — view source, and the markup is the documentation. Hierarchy comes from ink and weight, never from scale.

PLATE 01 · FOUNDATION

Color

Ten intent-named inks. The accent is yours to steer; everything else is the shared identity. Faint ink is decoration only — it never carries interactive meaning.

ink #151515 muted #737373 faint #a3a3a3 line #e9e9e9 accent #2643d0 ok #15714b warn #8a5f32 err #a84138

Each ink ships a light and a dark value; the toggle swaps the whole set in one 700ms breath. Radius 0 on every swatch — a square of the literal value.

SOURCE 
--ae-ink: #151515;  --ae-accent: #2643d0;
--ae-line: #e9e9e9; --ae-ok: #15714b;

PLATE 03 · FOUNDATION

Motion is feedback

Never decoration: small, brief, eased. State resolutions are gentler (480ms) and resolve exactly once — success persists, it is never rewound. The send button below resolves and stays resolved.

Under prefers-reduced-motion the same resolution is instant. Nothing changes size while it animates; nothing loops.

PLATE 06 · PRIMITIVE

Choice marks are squares

A circle would be the only curve in the system, so there isn't one. The checkbox fills with ink, the radio holds an ink core, the switch slides a thumb along a hairline channel — state reads from position and fill, never from a green pill.
