THE FEEL THE LAW REGISTERS SCREENS MOTION ADOPTION

THE FEEL

Instrument-panel restraint. Ink on paper: near-black text on paper-white, hairline rules, hard 90° corners, generous negative space, a Swiss-typographic grid. The page reads like a beautifully set technical document, never like a brochure.

Nothing glows, nothing floats, nothing bounces. One saturated ultramarine appears where it matters and nowhere else.

Margin notes ride in the chrome register at wide widths and fold away below 1100px.

THE LAW

One font size per surface; the 13px chrome register is the only exception. Hierarchy comes from ink and weight, never from scale. Nine registers: three inks by three weights.

The accent is rationed: exactly one instance per view — this document just spent its. Accents are never spent on buttons.

The law lives verbatim in the header of aesthetic.css and in tokens.json.

REGISTERS

Every line of text picks one of nine registers. The headline is heavier and blacker, not larger; support is muted; whispers are faint. 

--ae-ink:        #151515;
--ae-ink-muted:  #737373;
--ae-ink-faint:  #a3a3a3;
--ae-w-regular:  400;
--ae-w-medium:   550;
--ae-w-black:    800;

Code blocks sit in the wash — the only background a block of content is allowed to carry.

SCREENS

Screens, not pages: every view fits the viewport. This document is the sanctioned exception — long-form content scrolls inside the stage while the chrome never moves.

The bars above and below this text are fixed by the screen itself, not by position: fixed. Scroll and watch them hold.

The table of contents is sticky inside the scrolling stage and highlights as you pass each section.

MOTION

Motion is feedback, never decoration: small, brief, eased. State resolutions are gentler and resolve exactly once — success persists, it is never rewound. Nothing changes size while it animates, and reduced motion makes everything instant.

ADOPTION

One stylesheet, no build step. Pin a tag and upgrade deliberately: 

pnpm add github:misty-step/aesthetic#v2.3.1

Downstream projects may override exactly one token pair — the accent. Everything else is the shared identity.
