## New Features

- You can now reference all design tokens programmatically through the generated tokens.json file and browse them on the new token reference page
- You can now access interactive state matrices for all catalog primitives, showing hover, focus, disabled, and validation states
- You can now search the component catalog client-side for faster primitive discovery
- You can now integrate with the machine composition feed to automatically adopt design system updates in your tooling

## Improvements

- The catalog source blocks now include copy-to-clipboard functionality for easier code integration
