00 · AESTHETIC

One stylesheet is the whole system: ink on paper, hairline rules, hard corners, one font size on every surface. This page is its manual — eleven chapters scrolling inside the stage, every demonstration rendered by the file it documents. View source; the markup is the documentation.

01 · THE LAW

  • One size. Exactly one font size on every surface; the headline is heavier and blacker, never larger.
  • Nine registers. Three inks by three weights (400 / 550 / 800).
  • The accent is yours. Define your scheme and use it with judgment; identity comes from ink, hairlines, type, and motion. Color earns its place; it is not ambient.
  • Status rides the glyph. Danger, success, and warning color the icon — never the sentence, never a filled pill.
  • Screens, not pages. Every view fits the viewport; when content exceeds one screen, a quiet nav swaps views — or the document scrolls inside the stage, like this one.
  • Motion is feedback. Never decoration: small, brief, eased. State resolutions are gentler and resolve once — success persists, it is never rewound. Nothing changes size while it animates.
  • Buttons are not links. Contained ink shapes. Links navigate, buttons act.
  • Light and dark. Defaulting to the system, with a user toggle.

05 · CONTROLS

Settings rest as quiet hairline rows — label left, current value right — and fold a chooser open when asked. One row open at a time.

Choice marks are squares — a circle would be the only curve in the system, so there isn't one. The switch slides an ink thumb along a hairline channel; state reads from position and fill, never from a green pill.

Forms are lines, not boxes. The nearest field warms as the pointer approaches; sending resolves once and persists — it is never rewound.

09 · STEERING

Two strata. Invariants are never steered: one size, nine registers, hairlines, radius 0, motion as feedback, status on the glyph, buttons are not links. Dials are yours: the accent — one hue or several, named as project tokens — the status triplet, density, the mono ratio. A calm tool turns every dial down; a loud one turns up hue count and accent frequency — never radius, size scale, or ambient motion.
